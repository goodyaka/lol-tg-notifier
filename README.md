# LoL → Telegram Notifier

Фоновое приложение для **Windows**: следит за запуском League of Legends и
рассылает сообщение друзьям в Telegram через **Bot API**.

## Как это работает

1. В трее висит иконка. Приложение раз в несколько секунд проверяет, не
   появился ли процесс LoL.
2. Как только игра запускается — бот рассылает заданный текст всем
   получателям из `config.json`.
3. При закрытии игры «взводится» снова — следующий запуск тоже разошлёт.

Рассылка идёт **только тем, кто заранее нажал Start у твоего бота** — это
ограничение Telegram Bot API (бот не может писать первым незнакомым людям).

## Установка

```bash
pip install -r requirements.txt
cp config.example.json config.json   # создать свой конфиг (он не попадает в гит)
```

## Настройка (один раз)

1. **Создай бота**: напиши [@BotFather](https://t.me/BotFather) → `/newbot` →
   получи токен. Вставь его в `config.json` → `bot_token`.

2. **Собери chat_id друзей**:
   - попроси каждого из 10 друзей открыть твоего бота и нажать **Start**;
   - запусти сбор:
     ```bash
     python notifier.py --collect-ids
     ```
   - по мере нажатий будут появляться строки `+ Имя (chat_id=...)`;
   - нажми `Ctrl+C` — все найденные добавятся в `config.json`.

3. **Проверь** настройку и тестовую рассылку:
   ```bash
   python notifier.py --check       # токен + кол-во получателей
   python notifier.py --send-now    # реально разослать сейчас
   ```

4. **Настрой рассылку** в удобном окне:
   ```bash
   python notifier.py --settings
   ```
   В окне можно:
   - выбрать **режим игры** — 🏆 Ранкед / 💫 АРАМ / ⚔️ Арена (для каждого свой текст);
   - выбрать **кому слать**: «🦸 Avengers — общий сбор» (всем) или отметить
     галочками конкретных друзей;
   - нажать «Разослать сейчас» для теста.

   (Тексты режимов можно править и напрямую в `config.json` → `messages`.)

## Запуск

```bash
python notifier.py
```

Иконка появится в системном трее. Меню:
- **Настройки…** — окно с режимом игры и выбором получателей;
- **Рассылка активна** — вкл/выкл автоотправку;
- **Разослать сейчас** — ручная рассылка;
- **Выход**.

## Автозапуск вместе с системой (Windows)

Самый простой способ — положить ярлык в папку автозагрузки:

1. `Win + R` → `shell:startup` → Enter.
2. Создай в открывшейся папке ярлык на:
   ```
   pythonw.exe "C:\путь\к\lol-tg-notifier\notifier.py"
   ```
   (`pythonw.exe` — без чёрного окна консоли.)

После этого приложение стартует при входе в Windows и само поймает запуск LoL.

## Параметры config.json

| Поле | Значение |
|------|----------|
| `bot_token` | токен от @BotFather |
| `game_mode` | текущий режим: `ranked` / `aram` / `arena` |
| `send_to` | `all` (Avengers — всем) или `selected` (только отмеченные) |
| `messages` | тексты по режимам: `{ranked, aram, arena}` |
| `recipients` | список `{chat_id, name, selected}` (заполняется через `--collect-ids`) |
| `watch_processes` | имена процессов LoL для отслеживания |
| `poll_interval_sec` | как часто проверять процесс (сек) |
| `send_delay_sec` | пауза между сообщениями (бережёт от флуд-лимита) |
| `rearm_after_close` | слать снова при следующем запуске игры |
| `autostart_armed` | включена ли рассылка сразу при старте |

## Заметки

- Не превращай рассылку в спам: бот за частую отправку одинаковых сообщений
  может получить ограничения от Telegram. Для 10 друзей с паузой это не проблема.
- Если друг не получает сообщения — он не нажимал Start у бота либо нажал
  «стоп»/заблокировал бота.
